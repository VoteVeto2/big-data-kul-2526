Topic 1 Bagging vs. Boosting
1

Which statement best captures the core difference between bagging and boosting?

  1. Both train base learners sequentially, each correcting the previous one's errors.
  2. Bagging trains learners independently (in parallel) and mainly reduces variance; boosting trains them sequentially and mainly reduces bias.
  3. Bagging mainly reduces bias, while boosting mainly reduces variance.
  4. Bagging reweights misclassified instances, while boosting uses bootstrap samples.
2

Which statement correctly distinguishes AdaBoost from gradient boosting?

  1. AdaBoost fits each new learner to the residuals; gradient boosting reweights misclassified instances.
  2. AdaBoost increases the weights of misclassified instances; gradient boosting fits each new learner to the negative gradient (residuals) of the loss.
  3. Both are bagging methods that average independently trained deep trees.
  4. Both require fully grown (unpruned) trees as base learners.
Topic 2 t-SNE vs. UMAP
3

In t-SNE, which hyperparameter sets the effective size of each point's local neighborhood?

  1. Learning rate
  2. Perplexity
  3. min_dist
  4. Number of estimators
4

Which statement about t-SNE and UMAP is correct?

  1. Both are linear projection methods, like PCA.
  2. UMAP typically preserves global structure better and scales to large datasets faster than t-SNE.
  3. t-SNE can project new, unseen points into an existing embedding, whereas UMAP cannot.
  4. UMAP has no hyperparameters, while t-SNE has perplexity.
Topic 3 Lasso vs. Ridge
5

Which statement about Lasso (L1) and Ridge (L2) regression is correct?

  1. Ridge drives some coefficients exactly to zero, performing automatic feature selection.
  2. Lasso penalizes the sum of squared coefficients, while Ridge penalizes the sum of absolute values.
  3. Lasso can set some coefficients exactly to zero (feature selection), whereas Ridge only shrinks coefficients toward zero without eliminating them.
  4. Both methods always keep every feature with a non-zero coefficient.
6

You have many strongly correlated predictors and want a stable model that retains all of them. Which approach is most appropriate?

  1. Lasso (L1), because it keeps exactly one feature from each correlated group and zeros the rest.
  2. Ridge (L2), because it shrinks coefficients smoothly and spreads weight across correlated predictors, giving stable estimates.
  3. Ordinary least squares with no penalty, because regularization cannot handle correlation.
  4. Lasso (L1), because it assigns infinite weight to correlated features.
